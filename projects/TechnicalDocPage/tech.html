Introduction to HTML

HTML (HyperText Markup Language) is the standard language used to create web pages. It provides structure to web content by defining elements like headings, paragraphs, and links.

HTML documents are made up of elements enclosed in tags. These tags tell the browser how to display content.

Key Features of HTML:

  • Uses a simple tag-based syntax
  • Supports multimedia elements like images and videos
  • Works in combination with CSS and JavaScript

Example:

<!DOCTYPE html>
<html>
<head>
    <title>My First HTML Page</title>
</head>
<body>
    <h1>Welcome to My Page</h1>
    <p>This is a paragraph of text.</p>
</body>
</html>
Introduction to CSS

CSS (Cascading Style Sheets) is used to control the visual presentation of HTML elements. It allows developers to apply styles such as colors, fonts, and layouts.

CSS can be written inline, internally in a <style> tag, or externally in a .css file.

CSS Styling Techniques:

  • Inline styling using style attribute
  • Internal styles within a <style> tag
  • External stylesheets for better maintainability

Example

body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
    margin: 0;
    padding: 20px;
}
Introduction to JavaScript

JavaScript is a programming language used to create dynamic content and interactivity on websites. It can be used for form validation, animations, and fetching data from servers.

JavaScript can be written directly in an HTML document within <script> tags or in an external .js file.

Common JavaScript Features:

  • DOM Manipulation
  • Event Handling
  • Fetch API for HTTP Requests

Example

document.getElementById("demo").innerHTML = "Hello, JavaScript!";
Introduction to Git

Git is a version control system that allows developers to track changes in their codebase. It helps in collaboration and maintaining a history of modifications.

Git uses repositories to store project files and their revision history.

Basic Git Commands:

  • git init – Initialize a repository
  • git add .– Add changes to the staging area
  • git commit -m "message"– Commit changes with a message

Example:

git init
git add .
git commit -m "Initial commit"
Introduction to APIs

APIs (Application Programming Interfaces) allow different software applications to communicate with each other. They are widely used in web development to fetch or send data between services.

APIs can be categorized into RESTful APIs, SOAP APIs, and GraphQL APIs.

Common API Methods:

  • GET – Retrieve data
  • POST – Send data
  • PUT – Update existing data
  • DELETE – Remove data

Example:

{
    "user": "JohnDoe",
    "email": "johndoe@example.com"
}
